Password Hash Generator for Markata Password Protection Plugin

This script generates secure SHA-256 password hashes with salt for use with the password protection plugin.

Usage: python scripts/generate_password_hash.py python scripts/generate_password_hash.py --password "your_password" --salt "your_salt"

def generate_hash(password: str, salt: str = "blog_salt_2025") -> str:
    """Generate SHA-256 hash of password with salt."""
    combined = password + salt
    return hashlib.sha256(combined.encode('utf-8')).hexdigest()
